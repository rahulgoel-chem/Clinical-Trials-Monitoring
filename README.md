# Clinical-Trials-Monitoring
Clinical Trial Monitoring System using ClinicalTrials API and AACT database

This project monitors updates in clinical trials using:

- ClinicalTrials.gov API
- AACT Clinical Trials Database

Features:
- Detect new industry trials
- Detect trial updates
- Identify status, phase and enrollment changes
- Generate PDF intelligence reports
- Streamlit dashboard interface

Tech Stack:
Python, Streamlit, PostgreSQL, ReportLab
